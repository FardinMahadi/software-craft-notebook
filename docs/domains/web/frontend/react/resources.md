- [React Learn Guide](https://react.dev/learn) — official curriculum covering components, state, hooks, and effects.
- [Thinking in React](https://react.dev/learn/thinking-in-react) — structured approach to breaking UIs into components.
- [React Hooks Reference](https://react.dev/reference/react) — API reference for built-in hooks.
- [Beta React Docs: Effects and Refs](https://react.dev/learn/escape-hatches) — guidance on refs, effects, and escape hatches.
- [MDN JSX In Depth](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Spread_syntax#Spread_in_function_calls) — supplemental reference for JavaScript features used heavily in React apps.
