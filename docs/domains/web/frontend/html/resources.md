- [MDN HTML Guide](https://developer.mozilla.org/en-US/docs/Learn/HTML) — foundational tutorials and element references.
- [Inclusive Components](https://inclusive-components.design/) — practical accessibility patterns for forms, navigation, and interactive markup.
- [WebAIM Forms Checklist](https://webaim.org/techniques/forms/) — actionable form accessibility guidance.
- [W3C Tables Tutorials](https://www.w3.org/WAI/tutorials/tables/) — best practices for simple and complex data tables.
- [Landmark Roles Reference](https://www.w3.org/TR/wai-aria-practices/#aria_landmark) — clarifies when to rely on native elements versus ARIA.
