- [MDN CSS Guides](https://developer.mozilla.org/en-US/docs/Learn/CSS) — authoritative reference for core concepts.
- [Josh Comeau: CSS for JavaScript Developers](https://css-for-js.dev/) — deep dives on cascade and layout.
- [Every Layout](https://every-layout.dev/) — composable layout recipes with grid/flexbox.
- [Modern CSS Solutions](https://moderncss.dev/) — practical patterns for modern browser features.
- [SmolCSS](https://smolcss.dev/) — tiny examples demonstrating scalable CSS approaches.
