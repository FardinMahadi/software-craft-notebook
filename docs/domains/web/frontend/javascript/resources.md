- [MDN JavaScript Guide](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide) — authoritative language reference and tutorials.
- [JavaScript.info](https://javascript.info/) — in-depth explanations with practice tasks.
- [You Don’t Know JS Yet](https://github.com/getify/You-Dont-Know-JS) — deep dive into language mechanics.
- [React Docs: JavaScript Essentials](https://react.dev/learn/javascript-in-jsx-with-curly-braces) — highlights JS features used directly in React.
- [TC39 Proposals](https://github.com/tc39/proposals) — track upcoming language features relevant to modern tooling.
