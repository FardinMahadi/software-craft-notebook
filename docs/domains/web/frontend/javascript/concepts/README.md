# Learning Sequence

Study the concepts in order to build a foundation for React development:

1. `01-language-basics.md` — Variables, scope, types, and modern syntax.
2. `02-functions-closures.md` — Functions, `this`, closures, and higher-order patterns.
3. `03-objects-and-collections.md` — Immutable data handling with objects, arrays, maps, and sets.
4. `04-modules-and-tooling.md` — ES modules, bundlers, linting, and environment configuration.
5. `05-async-patterns.md` — Event loop, promises, async/await, fetch, and scheduling.

Complete related labs and checklist items after each concept to reinforce the topics.
