# Experiments

Lightweight spikes to test ideas quickly.

## Capture Format

- **Hypothesis** – what you expect to learn.
- **Setup** – tools, branches, environment.
- **Results** – measurements or qualitative takeaways.
- **Next Actions** – promote insights to `docs/` or plan a deeper project.

Archive completed spikes quarterly under `archive/projects/experiments/`.
