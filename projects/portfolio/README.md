# Portfolio

Curated work suitable for recruiters, collaborators, or public sharing.

## Checklist

- Clear narrative highlighting the problem, solution, and impact.
- Links to live demos or repositories (with setup instructions).
- Screenshots or recordings stored in `artifacts/`.
- Lessons summarised in `lessons-learned.md` and reflected in domain notes.

Review quarterly to keep the showcase aligned with current goals.
